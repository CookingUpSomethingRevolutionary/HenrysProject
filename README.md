# Cringy Sound Effects

This is a simple web page that showcases a collection of funny sound effect buttons using embedded Myinstants links.

## What this page includes
- A title section for the page heading
- An "About Me" section with a short introduction
- A scrollable soundboard area with embedded sound buttons

## Files
- `index.html` — the page structure
- `styles.css` — the styling for the layout, sections, and soundboard
